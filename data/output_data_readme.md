# Output Data

This directory contains final, ready-to-use datasets produced by the ETL pipeline.

## Output Structure
The output datasets are designed for analytics and modeling purposes and include:

- Housing NPL Dataset  
  - Joined with GDP Growth, Inflation Rate, Macro Shock Index, and MRR
- Automobile NPL Dataset  
  - Joined with GDP Growth, Inflation Rate, and Macro Shock Index
- Credit Card NPL Dataset  
  - Joined with GDP Growth, Inflation Rate, Macro Shock Index, and Credit Card Minimum Payment Rate

## Usage
- These datasets are used for:
  - Exploratory data analysis
  - Statistical modeling and forecasting
  - Reporting and dashboard development

## Notes
- Output data is excluded from the repository.
- The pipeline is designed to regenerate output datasets consistently from upstream stages.
